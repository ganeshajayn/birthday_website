# 🎂 Birthday Website — Assets

Place your files in this folder:

## Music
- `birthday-music.mp3` — Add a soft, romantic birthday song here.
  The music player will automatically use it.
  Recommended: royalty-free ambient piano music from pixabay.com or freemusicarchive.org

## Photos
- Add photos to this folder to display in the gallery.
  The gallery will use placeholder cards until real photos are added.
  Supported formats: JPG, PNG, WebP

## Uploads
- `uploads/` — Auto-created folder for photo uploads via the API.

---

**How to add music:**
1. Find a royalty-free MP3 file
2. Name it `birthday-music.mp3`
3. Place it in this `assets/` folder
4. Restart the server

The music player will show up automatically after the loading screen.
